設定

所有設定只儲存在這台瀏覽器上
影片/圖片會存進瀏覽器（IndexedDB），下次自動載入。建議影片 < 50MB。
Wallpaper Engine：
• 影片型（.mp4/.webm）→ 改用上方「本機檔案」選該影片。
• 網頁型（含 index.html）→ 在此貼上其 index.html 路徑。
• 編譯型 scene.pkg → 瀏覽器無法讀取，不支援。
預設自動偵測；若拒絕授權，改用下方固定城市。
類型：天氣＝自動天氣與穿搭；RSS＝貼上 feed 連結（PTT 看板用 https://www.ptt.cc/atom/看板.xml，可多個用空白分隔）； 連結＝常駐連結清單（每行「名稱 | 網址」）；F1＝賽事倒數與積分榜；番茄鐘＝專注／休息計時（可設分鐘數）。
RSS 兩道過濾：只顯示＝標題須含其一才顯示；排除＝標題含其一就剔除（例如排除「遠傳 電信 Apple Watch」）。
貼上 GitHub 權杖後，書籤／區塊／桌布等變更會自動存到你的 repo；任何電腦開這個網址都會自動載回（讀取免權杖）。 權杖只存在這台瀏覽器，請勿在公用電腦使用，可隨時到 GitHub 撤銷。 建立權杖 ↗ （Classic、勾選 public_repo 即可）
離線備份用：把目前設定存成 JSON 檔。跨電腦同步請用上方「雲端同步」，會自動把書籤與桌布帶到任何裝置。
每次儲存/匯入/雲端拉取前，系統會把上一份設定保留在本機（key: homepage_cfg_v3_backup），若不慎覆蓋可按「還原上一份」救回。
